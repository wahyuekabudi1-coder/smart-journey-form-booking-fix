# WeChat QR Code Assets

Silakan unggah file QR Code WeChat asli Anda di folder ini:
Nama file harus: **wechat-qr.png**
Path file: `/public/images/wechat-qr.png`

Anda dapat mengunggahnya langsung menggunakan File Explorer di sebelah kiri Google AI Studio dengan drag-and-drop atau klik kanan lalu klik "Upload".

Aplikasi akan otomatis membaca file tersebut dan menampilkannya di sistem pemesanan.
